The Famous Silver Mines

Silver was discovered in southern Arizona as early as 1858. During the beginning of Eds prospecting, he used an existing mine as a base to being his search called the San Pedro Mine. The abandoned camp was the site of 22 murders before Ed had arrived to prospect as the area is only 12 miles from hostile Apache camps. After finally locating float ore, it took him many more months to find the source. The vein was estimater to be 50 feet long and 12 inches wide. This news spread quickly throughout the Eastern US, which sparked a silver strike and growth of Tombstone plus 6 sister towns nearby. As the thousands of prospectors mined the San Pedro River, they eventually hit water at 520 ft below the surface, which flooded the mines. The solution was to pump water to surface tanks and utilize the water. This allowed them to continue to mine for a few more years, but eventually the mines flooded and ended the ability to mine silver ore. By 1930, Tombstone's population dropped to 150 people.

jQuery Methods


prev()- Get the immediately preceding sibling of each element in the set of matched elements. If a selector is provided, it retrieves the previous sibling only if it matches that selector.
find()-Get the descendants of each element in the current set of matched elements, filtered by a selector, jQuery object, or element.
attr()- Get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.
css()-Get the value of a computed style property for the first element in the set of matched elements or set one or more CSS properties for every matched element.
addClass()-Adds the specified class(es) to each element in the set of matched elements.
removeClass()-Remove a single class, multiple classes, or all classes from each element in the set of matched elements.
toggleClass()-Add or remove one or more classes from each element in the set of matched elements, depending on either the class’s presence or the value of the state argument.
html()-Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.
hide()-Hide the matched elements.
show()-Display the matched elements.
each()-Iterate over a jQuery object, executing a function for each matched element.
